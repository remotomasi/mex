# mex
Weather Forecast based on metcheck datas

### Prerequisites

* jq
* xvfb
* gnuplot 
* cutycapt

To install from bash command line, i.e.:
``` 
sudo apt-get install jq
```
Every sh and pg file must have the following grants: 755.
